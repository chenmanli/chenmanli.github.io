我的博客

第一篇博客想写一些心得和遇到的问题，刚开始搭建博客的时候我也是和大多数人一样是从网上搜的教程，
选择了hexo+github搭建静态博客，主要是因为hexo简单上手快，有利于前端新手学习。
问题一
在博客搭建过程中hexo init初始化的时候，没有在新建的Hexo文件夹中git bash，而是在桌面git快捷键中直
接hexo init的，所以当时初始文件都找不到，还急躁了好一阵，后来又来来回回看了几遍文档才发现自己错在何处。
问题二
在本地浏览博客时，浏览器输入localhost：4000，结果浏览器不能访问，上网查了一下，大部分的答案是让换端口号，
我尝试了一下没有成功，后来才发现原来是我没有打开服务器，在Hexo目录下git bash运行hexo s，浏览器上再次
输入localhost：4000，成功。
问题三
将博客部署到GitHub上时，报错Please tell me who you are.在网上查了资料后就在Hexo文件夹下git bash输入
git config –global user.email “you@example.com“
git config –global user.name “Your Name”，
但是结果还是报同样的错误，后来又看了几篇博客，才发现原来是目录出错了，要在.git这个隐藏目录下git bash，
然后输入git config –global user.email “you@example.com“
git config –global user.name “Your Name”，测试成功
